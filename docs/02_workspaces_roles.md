# Workspaces & Roles

Workspace roles:
- Owner: full control, including billing and workspace deletion.
- Admin: manage projects and members, but cannot delete workspace.
- Member: create and manage issues within assigned projects.
- Guest: read-only access to specific projects.

Inviting members:
- Owners and Admins can invite members via Workspace > Members > Invite.
- Invitations expire after 7 days.

Project visibility:
- Private projects: only invited members can access.
- Internal projects: anyone in the workspace can access.
