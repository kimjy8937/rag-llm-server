# DevBoard - Product Overview

DevBoard is a project and issue tracking SaaS for software teams.

Core concepts:
- Workspace: an organization-level container for projects.
- Project: a board with issues, labels, milestones, and attachments.
- Issue: a unit of work assigned to members.

Issue workflow:
- Default statuses: Backlog, In Progress, In Review, Done.
- Statuses can be customized per project.

Support routing:
- Login / tokens: see Account & Security.
- Roles / permissions: see Workspaces & Roles.
- Billing: see Billing & Plans.
- Refunds: see Refund Policy.
- Error codes: see Troubleshooting.
